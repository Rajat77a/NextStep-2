# Conversation Guide

The Conversation Guide page (`/parent/conversation`) surfaces the AI-generated conversation script from the most recent Clarity Check.

## Purpose

Parents often know their child needs support but don't know *how* to start the conversation. This page gives them a word-for-word script — warm, non-pressuring, and age-appropriate.

## Data Source

`ClarityCheck.conversationScript` — a long-form string generated by Claude as part of the report card analysis.

The prompt instructs Claude to write a script that:
- Opens with positive observations (even from yellow/red subjects)
- Asks open questions rather than making statements
- Avoids blame or comparison
- Ends with a concrete next step the parent and child can agree on together

## Page Structure

1. **Intro card** — brief explanation of how to use the script
2. **Script block** — the full conversation text, formatted with paragraph breaks
3. **Quick links** — buttons to Teacher Questions and 30-Day Plan

## Empty State

If no Clarity Check exists: prompt to upload a report card first.

## Tone Guidelines (for AI prompt authors)

The script must:
- Use "I noticed..." and "I was wondering..." instead of "Your grade shows..."
- Never use the word "failing" or "falling behind"
- Be under 300 words so the parent can read it in 2 minutes
- Include at least one specific subject mention (positive framing)
