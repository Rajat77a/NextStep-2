# NextStep.AI Documentation

This folder collects product, setup, deployment, and workflow notes for NextStep.AI.

The app is a Vite, React, and TypeScript project with separate experiences for parents, teachers, and school administrators. Data is currently handled in the browser with localStorage-backed API helpers, so these notes document the current implementation without assuming a production backend.
