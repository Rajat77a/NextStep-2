# Report Card Flow

The report card flow is the entry point for parent-facing guidance.

High-level steps:

1. Parent signs in or creates an account.
2. Parent uploads or enters report card details.
3. The app structures grades, comments, and observations.
4. A Clarity Check is generated from the report card context.
5. Follow-up guidance appears in related parent tools.

The flow should stay simple enough for a parent to complete without technical help.
