# Troubleshooting

## Deep Links Show 404

Confirm the SPA rewrite in `vercel.json` is present and deployed.

## Data Disappears

Check whether browser localStorage was cleared or a different browser/device is being used.

## Build Fails During Install

Check the lockfile registry URLs, Node version, and npm version.

## Build Fails During TypeScript

Run the build locally and address the first TypeScript error before redeploying.
